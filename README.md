ServiceNow ITSM Pre-Sales Playbook & Business Case
A complete avant-vente (pre-sales) cycle simulation for a ServiceNow ITSM deployment, built in Python on Google Colab. The scenario: Arvenne Industries SA, a fictional €870M French industrial manufacturer struggling with post-acquisition IT sprawl, an end-of-life GLPI instance, and production-line downtime exposure of €38k/hour.
Target role: Consultant Avant-Vente ServiceNow — this project reproduces the four artefacts a pre-sales consultant delivers in a real sales cycle.
